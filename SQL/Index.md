# Daftar Isi Modul SQL

Halaman ini berisi daftar modul pembelajaran SQL. Klik judul modul untuk membuka materi terkait.

## Yang Akan Dibahas

Halaman ini membahas urutan belajar SQL, daftar file materi, peta materi, dan target akhir pembelajaran. Gunakan halaman ini sebagai navigasi utama sebelum membuka setiap bagian.

## Tujuan Pembelajaran

Setelah membaca halaman ini, peserta diharapkan mampu:

1. Memahami alur belajar SQL dari dasar sampai lanjutan.
2. Memilih materi sesuai urutan yang disarankan.
3. Mengetahui target kemampuan yang perlu dicapai setelah menyelesaikan modul.

---

## Urutan Belajar yang Disarankan

1. [Pengenalan Database dan SQL](<pengenalan.md>)  
   Mengenal database relasional, tabel, baris, kolom, primary key, foreign key, SQL, dan kelompok perintah SQL.

2. [Membuat Database dan Tabel](<Membuat Database dan Tabel.md>)  
   Membahas DDL untuk membuat, mengubah, dan menghapus database serta tabel.

3. [Operasi CRUD](<Operasi CRUD.md>)  
   Mempelajari DML dan DQL dasar seperti `INSERT`, `SELECT`, `UPDATE`, dan `DELETE`.

4. [Filter, Sorting, dan Operator](<Filter Sorting dan Operator.md>)  
   Membahas `SELECT`, `WHERE`, operator perbandingan, operator logika, `BETWEEN`, `IN`, `LIKE`, `ORDER BY`, dan `LIMIT`.

5. [Aggregate Function dan Grouping](<Aggregate Function dan Grouping.md>)  
   Mempelajari `COUNT`, `SUM`, `AVG`, `MIN`, `MAX`, `GROUP BY`, dan `HAVING`.

6. [Relasi dan JOIN](<Relasi dan JOIN.md>)  
   Membahas relasi antar tabel serta penggunaan `INNER JOIN`, `LEFT JOIN`, `RIGHT JOIN`, `FULL OUTER JOIN`, dan JOIN lebih dari dua tabel.

7. [Constraint dan Integritas Data](<Constraint dan Integritas Data.md>)  
   Membahas `PRIMARY KEY`, `FOREIGN KEY`, `NOT NULL`, `UNIQUE`, dan `CHECK`.

8. [Subquery dan CTE](<Subquery dan CTE.md>)  
   Mempelajari subquery pada `WHERE` dan `FROM`, `EXISTS`, serta Common Table Expression.

9. [View](<View.md>)  
   Membahas pembuatan, penggunaan, dan penghapusan view.

10. [Window Function](<Window Function.md>)  
    Mempelajari `ROW_NUMBER`, `RANK`, `PARTITION BY`, dan running total.

11. [Transaction](<Transaction.md>)  
    Membahas TCL, transaction, `BEGIN`, `COMMIT`, `ROLLBACK`, dan prinsip ACID.

12. [Index dan Optimasi Query](<Index dan Optimasi Query.md>)  
    Membahas index, index pada foreign key, kapan menggunakan index, dan membaca query plan.

13. [Normalisasi Database](<Normalisasi Database.md>)  
    Membahas desain tabel yang lebih baik, 1NF, 2NF, dan 3NF.

14. [Stored Procedure dan Function](<Stored Procedure dan Function.md>)  
    Membahas contoh function PostgreSQL dan stored procedure MySQL.

15. [Kesalahan Umum Pemula](<Kesalahan Umum Pemula.md>)  
    Membahas kesalahan yang sering terjadi saat belajar SQL.

## Target Pembelajaran

Setelah menyelesaikan seluruh modul SQL, peserta diharapkan mampu:

- Memahami perbedaan DDL, DML, DQL, DCL, dan TCL.
- Membuat, mengubah, dan menghapus database serta tabel.
- Mengelola data menggunakan `INSERT`, `UPDATE`, dan `DELETE`.
- Mengambil data menggunakan `SELECT`, `WHERE`, sorting, grouping, dan aggregate function.
- Menggabungkan data dari beberapa tabel menggunakan JOIN.
- Menggunakan subquery, CTE, view, dan window function.
- Memahami transaction, index, normalisasi, dan dasar optimasi query.
