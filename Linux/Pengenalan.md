# 🐧 Modul Pembelajaran Linux: Pengenalan Linux

Linux adalah sistem operasi open source yang banyak digunakan di server, komputer pribadi, cloud computing, jaringan, keamanan siber, DevOps, dan pengembangan perangkat lunak. Linux dikenal stabil, aman, fleksibel, dan dapat digunakan secara gratis oleh siapa saja.

Modul ini dibuat sebagai materi awal sebelum mempelajari command dasar Linux. Peserta akan mengenal apa itu Linux, sejarah singkatnya, komponen utama, jenis distribusi, serta alasan Linux banyak digunakan di dunia teknologi.

## 1. Apa Itu Linux?

Linux adalah sistem operasi yang menghubungkan pengguna dengan perangkat keras komputer. Sistem operasi bertugas mengatur proses, memori, file, perangkat input-output, jaringan, dan aplikasi.

Secara sederhana:

```text
Pengguna -> Aplikasi -> Sistem Operasi Linux -> Perangkat Keras
```

Linux dapat digunakan melalui dua cara utama:

- **Graphical User Interface (GUI)**, yaitu tampilan visual seperti desktop, ikon, dan menu.
- **Command Line Interface (CLI)**, yaitu penggunaan perintah teks melalui terminal.

Contoh sistem operasi lain adalah Windows dan macOS. Perbedaan utamanya, Linux bersifat open source dan memiliki banyak variasi distribusi.

---

## 2. Sejarah Singkat Linux

Linux pertama kali dibuat oleh **Linus Torvalds** pada tahun 1991. Saat itu, Linus mengembangkan kernel sebagai proyek pribadi ketika ia masih menjadi mahasiswa.

Kernel Linux kemudian dirilis ke publik sehingga banyak pengembang dari seluruh dunia dapat ikut memperbaiki dan mengembangkannya.

Perkembangan penting Linux:

| Tahun | Peristiwa |
|---|---|
| 1991 | Linus Torvalds membuat kernel Linux |
| 1992 | Linux mulai menggunakan lisensi GNU GPL |
| 1990-an | Muncul berbagai distribusi Linux |
| 2000-an | Linux mulai banyak digunakan di server |
| Sekarang | Linux digunakan di server, cloud, Android, IoT, DevOps, dan cybersecurity |

---

## 3. Konsep Open Source

Linux adalah software **open source**, artinya kode sumbernya dapat dilihat, dipelajari, dimodifikasi, dan didistribusikan oleh siapa saja sesuai lisensi yang berlaku.

Keuntungan open source:

- Dapat digunakan secara gratis.
- Transparan karena kode sumber terbuka.
- Banyak komunitas yang ikut mengembangkan.
- Fleksibel untuk disesuaikan dengan kebutuhan.
- Cocok untuk pembelajaran teknologi.

Namun, open source bukan berarti tanpa aturan. Penggunaan dan distribusinya tetap mengikuti lisensi, misalnya GNU GPL.

---

## 4. Kernel dan Sistem Operasi

Kernel adalah inti dari sistem operasi. Kernel bertugas mengatur komunikasi antara software dan hardware.

Tugas kernel:

- Mengatur penggunaan CPU.
- Mengatur penggunaan memori.
- Mengelola file system.
- Mengatur perangkat seperti keyboard, mouse, storage, dan jaringan.
- Mengelola proses yang berjalan.

Linux sebenarnya adalah kernel. Namun dalam penggunaan sehari-hari, istilah Linux sering dipakai untuk menyebut sistem operasi lengkap yang berisi kernel, aplikasi sistem, package manager, shell, desktop environment, dan tools lainnya.

---

## 5. Distribusi Linux

Distribusi Linux, atau sering disebut **distro**, adalah sistem operasi lengkap yang dibangun menggunakan kernel Linux dan kumpulan software pendukung.

Contoh distro Linux populer:

| Distro | Keterangan |
|---|---|
| Ubuntu | Cocok untuk pemula, desktop, server, dan pembelajaran |
| Debian | Stabil dan banyak digunakan sebagai dasar distro lain |
| Linux Mint | Ramah untuk pengguna baru dari Windows |
| Fedora | Menggunakan teknologi baru dan cocok untuk developer |
| CentOS Stream | Sering digunakan untuk lingkungan server enterprise |
| Arch Linux | Fleksibel, ringan, dan cocok untuk pengguna tingkat lanjut |
| Kali Linux | Digunakan untuk cybersecurity dan penetration testing |

Setiap distro memiliki tujuan dan karakteristik yang berbeda. Untuk pemula, Ubuntu, Debian, atau Linux Mint sering menjadi pilihan yang nyaman.

---

## 6. Kelebihan Linux

Linux banyak digunakan karena memiliki beberapa kelebihan:

### Gratis dan Open Source

Sebagian besar distro Linux dapat digunakan tanpa biaya lisensi.

### Stabil

Linux dikenal stabil untuk penggunaan jangka panjang, terutama pada server.

### Aman

Linux memiliki sistem permission yang kuat dan komunitas aktif yang cepat memperbaiki celah keamanan.

### Ringan

Banyak distro Linux dapat berjalan di komputer dengan spesifikasi rendah.

### Fleksibel

Linux dapat digunakan untuk desktop, server, router, cloud, container, embedded system, dan perangkat IoT.

### Banyak Digunakan di Industri

Banyak server, layanan cloud, sistem DevOps, dan platform teknologi modern berjalan di Linux.

---

## 7. Kekurangan Linux

Linux juga memiliki beberapa tantangan, terutama bagi pengguna baru:

- Beberapa aplikasi populer Windows tidak tersedia secara langsung di Linux.
- Perlu adaptasi dengan terminal dan command line.
- Dukungan driver tertentu bisa berbeda tergantung perangkat.
- Beberapa game atau software khusus mungkin membutuhkan konfigurasi tambahan.

Meskipun demikian, Linux tetap sangat layak dipelajari karena menjadi fondasi penting dalam banyak bidang teknologi.

## 9. Tampilan Linux: GUI dan CLI

### GUI

GUI adalah tampilan visual yang dapat digunakan dengan mouse dan keyboard. Contohnya desktop environment seperti GNOME, KDE Plasma, XFCE, dan Cinnamon.

Kelebihan GUI:

- Mudah digunakan pemula.
- Tampilan lebih familiar.
- Cocok untuk pekerjaan desktop sehari-hari.

### CLI

CLI adalah antarmuka berbasis teks yang digunakan melalui terminal.

Kelebihan CLI:

- Cepat untuk administrasi sistem.
- Efisien untuk server.
- Mudah digunakan dalam otomasi.
- Banyak dipakai di dunia kerja teknologi.

Contoh command CLI:

```bash
pwd
ls
cd Documents
```

---

## 10. File System Linux

Linux memiliki struktur file system yang dimulai dari root directory `/`.

Beberapa direktori penting:

| Direktori | Fungsi |
|---|---|
| `/` | Direktori utama Linux |
| `/home` | Direktori user biasa |
| `/root` | Direktori user root |
| `/etc` | File konfigurasi sistem |
| `/bin` | Program command dasar |
| `/usr` | Aplikasi dan library tambahan |
| `/var` | Log dan data yang sering berubah |
| `/tmp` | File sementara |
| `/boot` | File booting sistem |

Contoh lokasi file:

```bash
/home/user/Documents/catatan.txt
```

---

## 11. User dan Permission

Linux mendukung banyak user dalam satu sistem. Setiap file dan folder memiliki pemilik serta hak akses.

Tiga jenis hak akses dasar:

| Permission | Arti |
|---|---|
| `read` | Izin membaca file |
| `write` | Izin mengubah file |
| `execute` | Izin menjalankan file |

Linux membagi hak akses untuk:

- **User**, yaitu pemilik file.
- **Group**, yaitu grup pemilik file.
- **Others**, yaitu pengguna lain.

Konsep permission ini membuat Linux lebih aman dan terkontrol.

---

## 12. Shell dan Terminal

Terminal adalah aplikasi untuk menjalankan perintah teks. Shell adalah program yang membaca dan menjalankan perintah tersebut.

Contoh shell:

- `bash`
- `zsh`
- `fish`
- `sh`

Contoh penggunaan terminal:

```bash
whoami
hostname
date
```

Dengan terminal, pengguna dapat mengelola file, menginstal aplikasi, memonitor sistem, mengatur server, dan membuat otomasi.

---

## 13. Package Manager

Package manager adalah alat untuk menginstal, memperbarui, dan menghapus aplikasi di Linux.

Contoh package manager:

| Distro | Package Manager |
|---|---|
| Ubuntu/Debian | `apt` |
| Fedora | `dnf` |
| Arch Linux | `pacman` |
| openSUSE | `zypper` |

Contoh pada Ubuntu/Debian:

```bash
sudo apt update
sudo apt install git
```

Package manager memudahkan pengguna mengelola software tanpa harus mengunduh installer manual dari banyak website.

Jika sudah memiliki Linux atau terminal Linux, coba jalankan command berikut:

```bash
whoami

pwd
ls
date

```

- WSL atau Windows Subsystem for Linux.
- VirtualBox dengan Ubuntu.
- Live USB Linux.
- Server VPS Linux.
- Platform cloud shell.
