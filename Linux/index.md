# 🐧 Daftar Isi Modul Linux

Halaman ini berisi daftar modul pembelajaran Linux. Klik judul modul untuk membuka materi terkait.

---

## Urutan Belajar yang Disarankan

1. [Pengenalan Linux](<Pengenalan.md>)  
   Mengenal Linux, sejarah, open source, distro, file system, user, shell, dan package manager.

2. [Basic Command Linux](<Basic Commad.md>)  
   Mempelajari command dasar seperti `pwd`, `ls`, `cd`, `mkdir`, `touch`, `cp`, `mv`, `rm`, `cat`, `grep`, dan command sistem lainnya.

3. [Users dan Group](<Users dan Group..md>)  
   Membahas user, group, root, sudo, pembuatan user, pengelolaan grup, dan praktik administrasi akun.

4. [File Permission](<File Permision.md>)  
   Membahas permission `read`, `write`, `execute`, mode simbolik, mode numerik, `chmod`, `chown`, dan `chgrp`.

5. [Package Manager](<Package Maneger.md>)  
   Membahas pengelolaan aplikasi menggunakan package manager seperti `apt`, `dnf`, `pacman`, dan konsep repository.

6. [Remote Server dan File Transfer](<Remote server dan File Transfer.md>)  
   Membahas akses server Linux menggunakan SSH serta transfer file menggunakan `scp`, `rsync`, dan `sftp`.

## Target Pembelajaran

Setelah menyelesaikan seluruh modul Linux, peserta diharapkan mampu:

- Memahami konsep dasar sistem operasi Linux.
- Menggunakan terminal untuk pekerjaan dasar.
- Mengelola file dan folder melalui command line.
- Mengatur user, group, dan permission.
- Menginstal dan menghapus aplikasi dengan package manager.
- Mengakses server Linux secara remote.
- Memindahkan file dari lokal ke server atau sebaliknya.
