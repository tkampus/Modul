## Target Pembelajaran

Setelah menyelesaikan seluruh modul, peserta diharapkan mampu:

- Memahami fungsi JavaScript dalam pengembangan web.
- Menginstal dan menggunakan Node.js serta npm.
- Menjalankan JavaScript di browser dan terminal.
- Menggunakan variabel, tipe data, operator, type conversion, percabangan, dan perulangan.
- Membuat function dan memahami scope.
- Mengolah array, object, string, number, date, JSON, dan sintaks modern seperti destructuring.
- Melakukan string manipulation untuk validasi, formatting, pencarian, pemotongan, dan pengubahan teks.
- Memahami class dan konsep OOP dasar.
- Membuat interaksi halaman web menggunakan DOM dan event.
- Menyimpan data sederhana menggunakan localStorage.
- Memahami asynchronous programming dengan promise dan `async await`.
- Menggunakan module dan package npm.
- Membaca, menangani, dan memperbaiki error dasar.
- Membuat mini project JavaScript sederhana.
